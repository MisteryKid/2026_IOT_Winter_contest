# Braker : oneM2M 기반 운전 자세 교정 시스템
---

# 서론
 본 프로젝트는 Arduino R4와 압력, 초음파 센서를 활용해 운전자의 발 위치를 실시간으로 모니터링하고, 올바르지 않은 운전 자세일 경우 사용자에게 알림을 주는 IoT 시스템이다. 

## 배경 
 최근 고령 운전자 증가와 함께 급발진 주장 사고가 사회적 이슈로 대두되고 있다. 이에 정부는 2029년부터 급발진 예방 장치 설치 의무화를 추진하고 있으나, 해당 장치는 정차중 오조작 방지에 특화되어 있어 실제 도로 주행 중 발생하는 페달 오인 사고를 완전히 막기에는 한계가 있다. 

 현재 보급되고 있는 페달 블랙박스 등은 사고 발생 후의 원인 규명에는 도움을 줄 수 있으나, 사고 자체를 미연에 방지하는 '예방적 기능'은 수행하지 못한다. 페달 오인 사고의 근본적인 원인 중 하나인 '운전 자세 불량'을 교정해 주는 능동적인 시스템이 부재한 실정이다. 

 따라서 본 프로젝트는 운전자가 평소에 올바른 운전자세를 유지하도록 피드벡을 주어 자세 불량으로 인한 페달 오인사고를 막고자 한다. 특히, oneM2M기반을 통해 수집한 센서 데이터를 통해 운전자의 평소 운전습관을 한눈에 보여주고자 한다.

## 핵심 기능
- 무릎 거리 센서를 통한 하체 위치 교정 피드백.
- 압력 센서를 통한 페달 오인 방지 및 뒤꿈치 위치 확인.
- oneM2M 표준 플랫폼을 활용한 센서 데이터의 안정적인 저장 및 웹 모니터링 연동.

## 기술 스택 
- Hardware: Arduino Uno R4 Wi-Fi (내장 Wi-Fi 모듈 활용)
- Sensors: 초음파 센서(무릎 거리 측정), 압력 센서(뒤꿈치 위치 확인)
- Platform: oneM2M (Mobius/Ocean) - 국제 표준 사물인터넷 플랫폼 활용
- Frontend: Vue.js
- Communication: HTTP/REST API (R4 Wi-Fi <-> oneM2M Server)
